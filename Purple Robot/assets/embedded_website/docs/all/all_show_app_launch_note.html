Displays a Purple Robot notification in the Android system bar that will launch the app named applicationName when tapped..

Arguments

title - Notification title.

message - Notification message.

applicationName - Name of the application to launch on tap.

displayWhen - Time expressed in Unix time milliseconds specifying when the notification should appear.

Return Values

True if the notification was scheduled or fired successfully, false if an error occured.

JavaScript Example

var now = Date.now(); // Time in milliseconds...

PurpleRobot.showLaunchNotification("Tap here to launch Facebook.", "If you tap this notification, Facebook will open on your device (if installed).", "Facebook", now);
